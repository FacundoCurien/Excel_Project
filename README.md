# Excel Project
An Excel project where I performed a **data cleaning**, a **data exploration** and a **data visualization** from a company board query.

### Tool used:
* Microsoft Excel

### The following Excel functions were used for this project:
* `Pivot Tables`.
* `Pivot Charts`.
* `Conditional Formatting`.
* `XLOOKUP`.
* `VLOOKUP`.
* `SUMIFS`.
* `COUNTIFS`.
* `LEN`.
* `MID`.
* `RIGHT`.
* `INDEX`
* `ROUND`.
* `Date Functions`.
